# 🏨 Hotel Booking System

A responsive Hotel Booking System built using **HTML, CSS, JavaScript, and Bootstrap 5**.  
This project provides a modern UI for hotel website landing pages with a navigation bar, image carousel, and structured layout.

---

## 🚀 Features

- Responsive Navigation Bar
- Bootstrap Image Carousel
- Modern UI Design
- Mobile Friendly Layout
- Clean and Structured Code
- Easy to Customize

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap 5
